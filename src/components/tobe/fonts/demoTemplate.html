第一步：使用font-face声明字体 
            @font-face {
                font-family: "blfont";
                src: url('./blfont.eot');
                src: url('./blfont.eot?#iefix') format('eot'),
                    url('./blfont.woff') format('woff'),
                    url('./blfont.ttf') format('truetype'),
                    url('./blfont.svg#blfont') format('svg');
            }
第二步：定义使用iconfont的样式 
.iconfont{
    font-family:"blfont" !important;
    font-size:16px;font-style:normal;
    -webkit-font-smoothing: antialiased;
    -webkit-text-stroke-width: 0.2px;
    -moz-osx-font-smoothing: grayscale;}
第三步：挑选相应图标并获取相应class，应用于页面,【iconfont-xxxx】=【iconfont-】前缀+【每个图标对应的class】 
<i class="iconfont gm-xxxx"></i>
